# Password Manager

## Opret vault

```bash
node manager.js create myMasterPassword
```

## Tilføj entry

```bash
node manager.js add myMasterPassword Email bruger@example.com MitPassword123
```

## Vis entry

```bash
node manager.js get myMasterPassword Email
```

## Søg entry

```bash
node manager.js search myMasterPassword Ema
```
